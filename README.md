# 📊 Global Sales & Performance Dashboard
An interactive Power BI Dashboard built using the Global Superstore Sales dataset to analyze worldwide 
sales performance, profitability, customer purchases, and regional trends. The dashboard enables users 
to explore key business metrics through dynamic visualizations, KPIs, and interactive filters, helping 
stakeholders make data-driven decisions.

#📌 Project Overview
The Global Sales & Performance Dashboard transforms raw transactional sales data into meaningful
business insights. It provides an intuitive interface for monitoring sales, profit, 
product performance, customer transactions, and regional performance.

The dashboard is designed for:

📈 Business Analysts
💼 Sales Managers
📊 Executives
📦 Operations Teams
🎓 Students learning Business Intelligence & Power BI

It enables quick analysis of business performance without manually exploring large datasets.


#3. 🚀 Tech Stack
The dashboard was developed using the following technologies:

📊 Power BI Desktop – Dashboard creation and visualization
🔄 Power Query – Data cleaning and transformation
🧠 DAX (Data Analysis Expressions) – KPIs, calculated columns, and measures
🔗 Data Modeling – Relationship creation between tables
📂 CSV Dataset – Global Superstore Sales Dataset
📁 File Formats
.pbix – Power BI Project
.png – Dashboard Preview
.pdf – Project Report.

#4. 📂 Data Source
Dataset: Global Superstore Sales Dataset

The project uses the Global Superstore Sales dataset from Kaggle, containing simulated real-world 
business transactions.

Dataset includes:
1.Customer ID
2.Customer Name
3.Product Name
4.Category
5.Sub-Category
6.Sales
7.Profit
8.Quantity
9.Shipping Cost
10.Region
11..Country
12.State
13.Order Date

To improve data modeling, an additional Categories table was created using DAX and linked with the
main dataset.

#5.🎯 Business Problem
Organizations generate thousands of sales transactions every day, making it difficult to
monitor performance using spreadsheets.

Business users often need answers to questions like:

1.Which region generates the highest sales?
2.Which product category performs best?
3.Is the business profitable?
4.How have sales changed over time?
5.Which customers contribute the most revenue?

Without an interactive dashboard, answering these questions requires significant manual effort.

The goal of this project is to provide a centralized dashboard that converts raw sales data into actionable insights.

#6. Dashboard Objectives

1. overall business performance
2. Track yearly sales growth
3. Compare sales across product categories
4. Analyze regional sales performance
5. Review customer-level transactions
6. Enable interactive filtering for faster analysis

#7. ✨ Dashboard Features

📌 KPI Cards

Displays overall business performance through key metrics:

💰 Total Sales
📈 Net Profit
📦 Total Quantity Sold
🚚 Total Shipping Cost

#8.🌍 Region Filter

Interactive slicer allowing users to filter the entire dashboard by region.

Example Regions:

Africa
Canada
Caribbean
Central Asia
East
North
EMEA

#9.🥧 Sales by Category

Donut chart comparing sales across:

Furniture
Office Supplies
Technology

Helps identify the highest revenue-generating product category.

#10.📈 Sales by Year
Line chart showing yearly sales trends.

Useful for:

Growth analysis
Trend identification
Year-over-year performance

#11.📊 Sales by Region

Bar chart highlighting sales generated by each region.

Enables comparison of regional business performance.

#12.📋 Individual Sales Breakdown
Detailed transaction table containing:

Customer ID
Customer Name
Product Name
Quantity
Region
Total Sales

Supports drill-down analysis at the transaction level.
